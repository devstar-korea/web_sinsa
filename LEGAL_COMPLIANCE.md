# 법적 준수사항 (Legal Compliance)

> ⚠️ **필독**: 이 프로젝트에서 반드시 지켜야 하는 법적 규칙입니다.
> 모든 개발자와 콘텐츠 기여자는 아래 규칙을 숙지하고 준수해야 합니다.

---

## 🚨 핵심 규칙

### 규칙 1: '평(坪)' 단위 사용 금지

**법적 근거**: 「표시·광고의 공정화에 관한 법률」 (표시광고법)

#### ❌ 금지사항
```typescript
// 잘못된 예시
<span>50평</span>
"면적: 30평"
title: "강남 50평 프리미엄 오피스"
```

#### ✅ 올바른 사용
```typescript
// 올바른 예시
<span>165㎡</span>
"면적: 99㎡"
title: "강남 165㎡ 프리미엄 오피스"
```

#### 📝 예외사항
- 매물 본문 설명(description)에서 참고용으로 언급은 가능
  - 예: "약 50평 규모의 넓은 공간" (보조 설명)
- **단, 주요 표기는 반드시 ㎡로만 표시**

#### 위반 시 처벌
- 과태료 최대 3,000만원
- 시정명령 불이행 시 영업정지

---

### 규칙 2: '보증금', '월세' 문구 사용 금지

**법적 근거**: 「공인중개사법」

#### ❌ 금지사항
```typescript
// 잘못된 예시
deposit: 3000만원
monthlyRent: 150만원
"보증금 3,000만원"
"월세 150만원"
```

#### ✅ 대체 표현
```typescript
// 올바른 예시
totalInvestment: 8000만원  // 총 투자비용
premiumAmount: 5000만원   // 권리금
monthlyProfit: 450만원    // 월수익

// 내부 관리용 (비공개)
// 관리자 페이지에서만 사용 가능, 프론트엔드 노출 금지
```

#### 사용 가능한 용어
- ✅ 권리금 (Premium)
- ✅ 총 투자비용 (Total Investment)
- ✅ 월수익 (Monthly Profit)
- ✅ 매매가 (Sale Price)

#### 위반 시 처벌
- 공인중개사법 위반
- 무등록 중개업 행위로 간주
- 벌금형 또는 징역형 (3년 이하)

## 📋 준수 체크리스트

### 개발 시 확인사항

- [ ] **UI 컴포넌트**
  - [ ] ListingCard에 '평' 표시 없는지 확인
  - [ ] 매물 상세 페이지에 '평' 표시 없는지 확인
  - [ ] 필터/정렬 옵션에 '평' 사용 안 함

- [ ] **타입 정의**
  - [ ] `deposit`, `monthlyRent` 필드가 프론트엔드 타입에 없는지 확인
  - [ ] Admin 전용 타입에만 존재하는지 확인

- [ ] **API 응답**
  - [ ] 공개 API에서 보증금/월세 정보 노출 안 되는지 확인
  - [ ] 관리자 API에만 민감 정보 포함되는지 확인

- [ ] **더미 데이터 / 실제 데이터**
  - [ ] 면적은 ㎡로만 저장
  - [ ] 보증금/월세는 비공개 필드로만 관리

## 🔒 개인정보보호 규칙

### 규칙 3: 상세 위치 정보 비공개

**법적 근거**: 「개인정보보호법」, 매도자 프라이버시 보호

#### ❌ 금지사항
```typescript
// 잘못된 예시
city: '강남구'
displayLocation: '강남역 3분'
address: '서울시 강남구 테헤란로 123'
```

#### ✅ 올바른 사용
```typescript
// 올바른 예시
province: '서울'
locationKey: '서울'

// 내부 관리용 (비공개)
// 구체적 위치는 관리자 페이지에만 저장
```

#### 이유
- 사진 정보와 결합 시 매물 특정 가능
- 매도자가 노출을 원하지 않음
- 거래 정보 유출 방지

## 🛡️ 기타 준수사항

### 허위·과장 광고 금지
- 사실과 다른 정보 게시 금지
- 검증되지 않은 수익률 표시 금지
- 매물 상태를 과장하여 표현 금지

### 가격 표시 원칙
- 협의가능 여부 명확히 표시
- 부가세 별도 여부 표시 (해당 시)
